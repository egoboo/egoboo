Controls
========
This file lets users modify the handling of input devices.
See the game manual for a list of settings and more info.
Note that you can mix KEY_ type settings with other 
devices... Write the input after the colons!

General Controls
========
These are general controls and cannot be changed
ESC                   - End module
SPACE                 - Respawn character (if dead and possible)
1 to 7                - Show character detailed stats
LEFT SHIFT   + 1 to 7 - Show selected character armor without magic enchants
LEFT CONTROL + 1 to 7 - Show armor stats with magic enchants included
LEFT ALT     + 1 to 7 - Show character magic enchants
F11                   - Take screenshot

Up			: KEY_PAD_8
